# 工具输出优化

## 描述

提供工具相关的管理脚本，包括数据清理、信息收集和配置验证。

## 脚本

### clear_execution_data.py
清空数据库中的执行记录和任务数据。

### collect_tool_info.py
收集所有工具信息并生成配置文件。

### validate_tool_config.py
验证工具代码定义与配置文件是否一致。

## 使用场景

- 清理测试环境数据
- 收集和更新工具配置
- 验证工具配置一致性
