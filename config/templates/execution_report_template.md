## 风险分级标签 [必填]

> 编排器根据本标签决定是否派独立审查。判定权在执行 Agent（它知道自己改了什么）。
> 任一维度命中 → independent_review_needed=true。
> 纯函数内部逻辑/单模块实现/测试代码/注释文档/不改接口的性能优化 → false。

- **是否需要独立审查(independent_review_needed)**: true / false
- **命中维度**（命中才勾，全不命中则此处留空）:
  - [ ] 接口契约变更（新增/修改/删除对外接口、函数签名、数据结构、配置项）
  - [ ] 跨模块改动（涉及 ≥2 个模块，非同模块多文件）
  - [ ] 安全相关（认证、鉴权、加密、密钥、权限、用户数据）
  - [ ] 数据持久化（数据库schema、配置文件结构、状态存储）
  - [ ] 控制流核心（主循环、状态机、任务调度、错误处理策略）
