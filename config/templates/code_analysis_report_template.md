## 评估指南

### 检查维度

| 维度 | 类型 | 检查内容 |
|------|------|----------|
| 完整性 | 必填 | 所有 [必填] 章节已填写，问题清单完整 |
| 准确性 | 必填 | 问题定位准确（有文件路径和行号），严重程度分级合理 |
| 客观性 | 必填 | verdict 基于规则和自动化工具客观结果，禁止主观通融 |

### 评估结论

> | 结论 | 判定条件 | 后续动作 |
> |------|---------|---------|
> | ✅ Approve | 物理保险全过 + 无 Must Fix | 任务完成 |
> | ❌ Request Changes | 物理保险有失败 或 存在 Must Fix | 回退修复 |
