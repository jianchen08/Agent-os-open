# 主题自定义使用指南

本指南介绍如何使用系统的主题功能，包括切换主题、创建自定义主题、编辑组件样式等。

---

## 1. 主题切换基础

### 1.1 快速切换深色/浅色模式

系统提供三种主题模式：

- **浅色模式**：适合白天使用，界面明亮
- **深色模式**：适合夜间使用，护眼
- **跟随系统**：自动根据操作系统设置切换

**操作方式**：

1. 点击顶部导航栏的主题按钮（太阳/月亮图标）
2. 在主题面板中选择想要的模式

### 1.2 切换预设主题

系统内置多个预设主题：

- **深色主题**：默认深色配色
- **浅色主题**：默认浅色配色
- **更多主题**：系统可能会添加更多预设主题

**操作方式**：

1. 打开「设置」→「主题」
2. 在「预设主题」列表中选择想要的主题
3. 点击「应用」按钮

---

## 2. 创建自定义主题

### 2.1 基于预设主题创建

最简单的方式是基于现有主题进行修改：

1. 打开「设置」→「主题」→「主题管理」
2. 在预设主题列表中，点击「复制」按钮
3. 系统会创建一个新主题，包含原主题的所有设置
4. 修改新主题的样式
5. 保存时输入主题名称

### 2.2 从空白创建

1. 点击「创建新主题」按钮
2. 选择基础主题（建议选择一个接近你想要的预设主题）
3. 系统会创建一个空白主题，继承基础主题的配置
4. 开始编辑你的主题

---

## 3. 组件样式编辑

### 3.1 编辑按钮样式

你可以单独控制每种按钮变体的样式：

**可编辑的按钮类型**：
- **Primary 按钮**：主要操作按钮，通常使用主色调
- **Secondary 按钮**：次要操作按钮，使用次要色
- **Ghost 按钮**：幽灵按钮，透明背景
- **Destructive 按钮**：危险操作按钮，使用警告色

**可编辑的属性**：
- 背景颜色
- 文字颜色
- 边框颜色和宽度
- 悬停效果颜色
- 阴影效果
- 圆角大小

**操作方式**：
1. 在主题编辑器中找到「按钮样式」部分
2. 选择要编辑的按钮类型
3. 使用颜色选择器修改颜色
4. 实时预览效果

### 3.2 编辑消息气泡样式

消息气泡分为用户消息和 AI 消息，可以独立控制样式：

**用户消息气泡**：
- 背景颜色（通常使用主色调）
- 文字颜色
- 圆角大小
- 右上角圆角（特殊处理，形成对话气泡效果）

**AI 消息气泡**：
- 背景颜色（通常使用中性色）
- 文字颜色
- 圆角大小
- 左上角圆角（特殊处理）

**操作方式**：
1. 在主题编辑器中找到「消息气泡」部分
2. 选择「用户消息」或「AI 消息」
3. 调整颜色和圆角
4. 右侧预览区会实时显示效果

### 3.3 编辑头像样式

**可编辑属性**：
- 圆角大小（方形、圆角、圆形）
- 边框颜色和宽度
- 阴影效果
- 尺寸（小、中、大）

**操作方式**：
1. 在主题编辑器中找到「头像样式」部分
2. 调整各项属性
3. 预览区会显示不同尺寸的头像效果

### 3.4 编辑卡片样式

**可编辑属性**：
- 背景颜色
- 边框颜色和宽度
- 圆角大小
- 阴影效果
- 毛玻璃效果（模糊程度）

**操作方式**：
1. 在主题编辑器中找到「卡片样式」部分
2. 选择卡片样式类型（平面、凸起、描边、毛玻璃）
3. 调整各项属性

## 4. 实时预览

### 4.1 预览模式

主题编辑器提供两种预览模式：

- **实时预览**：修改后立即应用到整个界面
- **冻结预览**：只在预览区域显示效果，不影响主界面

**切换方式**：
点击编辑器顶部的「预览模式」按钮

### 4.2 组件预览

预览区域包含以下组件：

- 按钮组（展示所有按钮变体）
- 消息气泡（用户和 AI 消息示例）
- 头像组（不同尺寸的头像）
- 卡片示例（展示不同样式）

### 4.3 撤销和重做

编辑器支持撤销/重做功能：

- **撤销**：点击工具栏的撤销按钮（或按 `Ctrl+Z`）
- **重做**：点击工具栏的重做按钮（或按 `Ctrl+Shift+Z`）

## 5. 保存和管理主题

### 5.1 保存主题

编辑完成后：

1. 点击「保存」按钮
2. 输入主题名称
3. 选择是否设为当前主题
4. 点击「确认」

主题会保存到浏览器的 localStorage 中。

### 5.2 管理已保存的主题

在「主题管理」页面可以：

- 查看所有已保存的自定义主题
- 编辑现有主题
- 删除不需要的主题
- 导出主题配置文件
- 导入他人分享的主题配置

### 5.3 导出主题

将你的主题分享给他人：

1. 在主题列表中找到要导出的主题
2. 点击「导出」按钮
3. 系统会生成一个 JSON 配置文件
4. 保存到本地或分享给他人

### 5.4 导入主题

导入他人分享的主题：

1. 在「主题管理」页面点击「导入」
2. 选择主题配置文件（JSON 格式）
3. 系统会验证配置格式
4. 确认导入

## 6. 高级功能

### 6.1 继承机制

自定义主题继承自某个预设主题，只包含你修改过的部分。这样的好处是：

- 减少配置文件大小
- 当预设主题更新时，自定义主题也会受益
- 更容易管理和维护

**注意**：如果删除了基础预设主题，基于它的自定义主题也会受影响。

### 6.2 效果配置

除了颜色和样式，还可以配置：

- **毛玻璃效果**：开启后卡片会有半透明模糊效果
- **动画**：开启后界面会有过渡动画
- **减少动画**：为喜欢简洁界面的用户设计
- **过渡时长**：调整动画速度

### 6.3 无障碍配置

主题支持无障碍功能：

- **高对比度**：增强文字和背景的对比度
- **焦点指示器**：突出显示当前聚焦的元素
- **减少动画**：减少动画效果，适合容易晕眩的用户
- **大字体**：增大字体尺寸

## 7. 常见问题

### Q1: 为什么我的主题没有保存？

**A**: 主题保存在浏览器的 localStorage 中。如果：

- 使用了隐私/无痕模式
- 清除了浏览器缓存
- 更换了浏览器或设备

主题可能会丢失。建议定期导出重要主题作为备份。

### Q2: 如何恢复默认主题？

**A**:
1. 打开「设置」→「主题」
2. 选择预设主题中的「深色主题」或「浅色主题」
3. 点击「应用」

如果需要完全重置：
1. 在「主题管理」中删除所有自定义主题
2. 刷新页面

### Q3: 自定义主题会影响性能吗？

**A**: 不会。主题配置在加载时应用，之后不会有额外的性能开销。CSS 变量经过优化，切换主题非常快。

### Q4: 可以在移动设备上使用吗？

**A**: 可以。主题响应式设计，在手机、平板、电脑上都能正常使用。

### Q5: 如何分享我的主题？

**A**:
1. 导出主题配置文件
2. 将 JSON 文件分享给他人
3. 对方通过「导入」功能即可使用

### Q6: 主题配置占用多少空间？

**A**: 一个典型主题配置约 2-5 KB，存储在 localStorage 中。即使保存几十个主题，也不会占用太多空间。

## 8. 最佳实践

### 8.1 配色建议

- **主色调**：选择一个代表你品牌的颜色
- **对比度**：确保文字和背景有足够的对比度（建议 4.5:1 以上）
- **一致性**：保持整体风格统一，避免使用太多颜色
- **可读性**：优先考虑文字的可读性

### 8.2 主题命名建议

- 使用描述性的名称，如「深蓝专业版」「柔和护眼版」
- 可以包含用途说明，如「夜间工作」「演示专用」
- 避免使用特殊字符

### 8.3 备份建议

- 定期导出重要主题
- 保存到云存储或其他安全位置
- 为不同场景创建不同主题（工作、演示、个人）
